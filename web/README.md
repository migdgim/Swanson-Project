# web/ — interfaccia pubblica (Next.js)

Costruita in **S5**, dopo che il time-slicing (S2) ha dato esito positivo.

Servirà il sottoinsieme **curato e validato** delle ipotesi:
- esploratore del grafo (Cytoscape.js / Sigma.js, dati da API),
- pagine-ipotesi con le evidenze (PMID a supporto),
- il report di validazione (metriche, falsi positivi) **in chiaro**, così la scienza resta trasparente.

Backend: **Supabase (Postgres)** — vedi [`../supabase/`](../supabase/). Hosting previsto: Vercel.
